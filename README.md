Ultra Minimal Weather

Single-page demo that shows uses async API calls in JavaScript.




Open `index.html` in any modern browser.
Type a location (e.g., `Navojoa, MX`) and press Enter.


Features
    - Open-Meteo Geocoding API → converts text into latitude/longitude.
    - Open-Meteo Forecast API → fetches current weather.
    - Async/await flow with friendly error handling for typos, bad matches, and offline mode.
